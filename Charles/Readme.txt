DuckyText: Usage ". ./duckyText [command]" The first . resources the bashrc
TouchNoise: Usage "./touchNoise [path to recursivly touch]"
HashFind: Usage "./hashFind [md5hash]" or "./hashFind `md5hash [file]`"

DuckyText adds an alias to the bashrc that will cause the command in question to have random characters appened to the command
TouchNoise will recursivly touch files in a given path
HashFind will return the file location of a file with a particular md5hash
